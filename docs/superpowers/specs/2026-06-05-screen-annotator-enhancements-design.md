# 桌面批注工具增强设计规范

## 1. 设计目标
本规范定义当前桌面批注工具的下一阶段增强方案，重点覆盖三类需求：

1. 现有工具栏 UI 与交互稳定性修复
2. 画笔辅助模式（规则图形修正）
3. 字体自识别模式（连续手写识别与文本替换）

本次增强遵循以下原则：

- 优先修复会直接影响使用体验的鼠标样式抽搐、边缘效应和工具栏位移问题
- 视觉风格保持简洁、克制，不使用虚假的高光、镜面反射或过强渐变
- 智能能力必须先给出稳定边界，再逐步扩展，不以牺牲基础绘制体验为代价
- 所有新能力都必须与现有撤回、清空、橡皮和主题切换兼容

## 2. 总体分层方案

### 2.1 分阶段落地顺序
本次增强按三层实施，但设计一次成型：

1. **UI/交互稳定层**
2. **辅助模式层**
3. **字体自识别层**

### 2.2 结构职责

#### Toolbar Window
- 负责悬浮球、主工具栏、主题切换、设置面板、模式开关
- 不承载任何识别逻辑，只负责交互意图发起和参数配置

#### Canvas Window
- 负责普通自由绘制、辅助模式预览、规则图形对象渲染、文本对象渲染
- 保持全屏透明绘制层的角色，不承担工具栏布局职责

#### Recognition Layer
- 独立处理轨迹理解与识别
- 分为：
  - `shape recognition`: 图形辅助修正
  - `text recognition`: 连续手写文本识别

#### State Layer
- 在现有画布状态模型之上扩展统一对象栈
- 对象类型至少包含：
  - 普通笔迹对象
  - 辅助图形对象
  - 文本对象
- 所有对象统一接入撤回、清空和重绘流程

## 3. UI 与交互规范

### 3.1 悬浮球
- 尺寸较当前版本再放大一档，保持小巧但更易命中
- 保留左上到右下的深色分区风格
- 深色模式使用 `rgb(20, 19, 20)` 与纯黑的低对比过渡
- 不使用光泽感、镜面高光或“AI 高亮”
- 必须保留背后阴影，阴影需要稳定、简洁，不发蓝光
- 鼠标移入悬浮球时：
  - 光标变为手形
  - 球体轻微放大
  - 阴影略增强

### 3.2 主工具栏
- 主工具栏继续采用垂直布局
- 工具栏竖向高度提升，横向保持较窄，不再回到“太胖”的视觉
- 工具顺序调整为：
  1. 鼠标
  2. 笔
  3. 橡皮
  4. 撤回
  5. 清屏
  6. 明暗模式
  7. 字体自识别
  8. 收起
- 高亮态使用浅白色块与细边框，不使用蓝色光效
- 主题按钮图标按当前模式切换：
  - 深色模式显示月亮
  - 浅色模式显示太阳

### 3.3 蒙版区与鼠标事件
- 禁止大面积透明交互热区
- 只有真实可见的工具栏本体和设置面板允许接管鼠标事件
- 主工具栏和设置面板分别独立命中，不允许再依赖额外透明蒙版区兜底
- 穿透状态只在以下时机切换：
  - 鼠标进入真实可见控件
  - 鼠标离开全部真实可见控件
- 必须继续保留一个很短的离开缓冲时间，用于降低边缘区域抖动
- 目标效果：
  - 工具栏区域外不抢画布光标
  - 工具栏区域内不暴露底层原始光标
  - 设置弹窗与主栏之间不再出现样式抽搐

### 3.4 设置面板
- 再次点击已高亮按钮时，主工具栏本体保持固定位置
- 设置面板从固定锚点侧滑展开
- 设置面板使用绝对定位，不参与主栏流式布局
- 动画保留，但幅度减小：
  - 轻微位移
  - 轻微透明度变化
  - 不使用夸张 blur 或大幅缩放

### 3.5 主题视觉

#### 深色模式
- 主色基底为纯净深灰黑
- 无高光、无玻璃反射、无亮面条纹
- 滑块和按钮保持低对比但足够可辨

#### 浅色模式
- 主色基底为浅灰白
- 与深色模式采用相同层级结构，但整体更柔和
- 不引入 glossy 质感

### 3.6 滑块样式
- 滑块改为“内嵌式”结构
- 轨道为细圆角槽体
- 滑块 thumb 看起来像嵌在槽内，而不是悬浮圆球
- 深色模式：
  - 槽体偏深
  - thumb 偏浅白
- 浅色模式：
  - 槽体偏浅灰
  - thumb 偏浅灰白
- 滑块在视觉上必须简洁、平滑、克制

### 3.7 橡皮设置
- 继续保留橡皮大小设置
- 在橡皮设置面板中新增 `清除所有`
- `清除所有` 作为橡皮上下文中的高级清理动作，不放入主栏按钮区

### 3.8 画笔设置
- 保留现有颜色、大小、轨迹长度、轨迹颜色
- 新增一个标准 RGB 取色板：
  - 允许用户通过 `R/G/B` 三个 `0-255` 数值输入自定义颜色
  - 同步提供颜色预览块
- 新增 `辅助模式` 开关：
  - 关闭：自由绘制
  - 开启：图形辅助修正

## 4. 字体自识别模式

### 4.1 触发方式
- 仅在用户切换到 `字体自识别` 工具时启用
- 用户书写期间保留原始笔迹采样
- 用户停止书写后等待 `3 秒`
- `3 秒` 内若没有继续写，则把最近一段连续笔迹视为一个识别批次

### 4.2 分组策略
- 不做逐字弹窗确认
- 识别单位优先是：
  - 词组
  - 连续短句
  - 单行连续内容
- 分组依据：
  - 笔迹时间间隔
  - 空间连续性
  - 基线接近程度
- 若笔迹间隔明显拉长或位置跳跃过远，则新建下一组

### 4.3 确认方式
- 识别后显示轻量确认条，而不是逐字确认面板
- 确认条包含：
  - 主识别结果
  - 推荐字号
  - `确认替换`
  - `保留手写`
  - `重新识别`
- 一次确认作用于整组识别结果

### 4.4 替换规则
- 用户确认后，将原手写笔迹替换为文本对象
- 第一版字体固定为微软字体体系：
  - 中文优先 `Microsoft YaHei`
- 字号由原始笔迹包围盒高度估算
- 替换后的文本对象加入统一历史栈，支持撤回和清空

### 4.5 失败回退
- 如果识别置信度不足，不自动替换
- 系统提示“未可靠识别，保留手写”
- 不允许在低置信度下直接生成错误文本

## 5. 辅助模式

### 5.1 开启方式
- 辅助模式作为画笔设置中的一个开关，不作为独立主工具
- 开启后，用户仍使用画笔绘制，但系统尝试识别并修正形状

### 5.2 基本交互
- 用户按住左键绘制图形
- 系统持续采样当前轨迹
- 在用户仍按住左键时，实时显示“修正后的预览形状”
- 用户松开左键后，提交最终图形对象

### 5.3 识别边界
“自由图形都支持”在第一版中的定义为：所有轨迹都允许进入尝试识别流程，但只对高置信度图形执行自动修正。

#### 高置信度自动修正
- 直线
- 圆
- 椭圆
- 矩形
- 圆角矩形
- 箭头

#### 中置信度仅预览不强制修正
- 波浪线
- 多边形
- 近似闭合自由图形

#### 低置信度保持原手绘
- 涂鸦
- 明显不规则自由图形
- 无法归类的复杂轨迹

### 5.4 判定方法
- 直线：整体方向一致性 + 点到拟合直线偏差
- 圆/椭圆：闭合度 + 曲率稳定性 + 长短轴分布
- 矩形：角点聚类 + 四段方向分布 + 闭合度
- 箭头：主干线段 + 末端分叉结构

### 5.5 结果对象
- 规则图形作为独立图形对象进入对象栈
- 后续必须兼容：
  - 撤回
  - 清空
  - 重绘

## 6. 数据模型扩展

### 6.1 建议新增对象类型
- `stroke`: 普通自由笔迹
- `shape`: 辅助模式生成的规则图形
- `text`: 字体自识别生成的文本对象

### 6.2 建议新增运行态状态
- 当前主题模式
- 当前是否启用辅助模式
- 当前是否启用字体自识别模式
- 当前待识别笔迹批次
- 当前图形预览对象
- 当前文字确认条状态

## 7. 测试策略

### 7.1 单元测试
- 工具栏模型：
  - 工具顺序
  - 主题按钮标签与状态
  - 悬浮球尺寸
  - 工具栏尺寸
- 鼠标命中模型：
  - 透明区穿透
  - 真实控件命中
  - 离开缓冲
- 图形识别模型：
  - 直线、圆、矩形、箭头拟合与置信度
- 文本分组模型：
  - 3 秒停笔触发
  - 连续笔迹合并
  - 长间隔与远距离拆组

### 7.2 集成测试
- 画笔普通模式与辅助模式切换
- 橡皮、撤回、清除所有与对象栈兼容
- 文本识别确认后替换为文本对象
- 设置面板展开时主工具栏不位移

### 7.3 运行态验证
- 工具栏区域外不抢鼠标样式
- 设置面板区域不出现边缘抽搐
- 深浅色模式切换稳定
- 3 秒后整组识别触发正确

## 8. 风险与控制

### 8.1 蒙版区仍有边缘抖动
- 控制策略：进一步收窄交互命中范围，只保留真实可见控件命中

### 8.2 辅助模式误判过多
- 控制策略：只自动提交高置信度图形，中低置信度保留手绘

### 8.3 文字识别误替换
- 控制策略：坚持整组确认后替换，不做无确认自动替换

### 8.4 对象类型增多后历史栈混乱
- 控制策略：统一对象栈模型，不为文本和图形单独维护一套撤回系统

## 9. 实施顺序
1. 修复 UI/交互稳定问题
2. 完成画笔设置增强、RGB 取色板、橡皮 `清除所有`
3. 完成辅助模式基础预览与高置信度图形修正
4. 完成字体自识别的 3 秒分组、统一确认与文本替换
5. 补齐自动化测试与运行态验证

## 10. 验收标准
- 悬浮球更大、带阴影、悬停变手形
- 工具栏蒙版区不再出现鼠标样式抽搐
- 工具栏更高，但仍保持简洁窄型
- 高亮为浅白风格，不带蓝色光效或虚假高光
- 明暗模式使用月亮/太阳图标
- 滑块视觉为内嵌式简洁风
- 橡皮设置中可直接执行 `清除所有`
- 字体自识别可在停笔 3 秒后按整句/词组确认替换
- 辅助模式能将高置信度手绘修正为规则图形
- 所有新增对象兼容撤回、清空与主题切换
